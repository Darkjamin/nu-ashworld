# NuAshworld

[NuAshworld](http://nu-ashworld.lamdera.app) is a [source-available](https://github.com/Janiczek/nu-ashworld) multiplayer turn-based [browser game](https://en.wikipedia.org/wiki/Browser_game) set in the universe of [Fallout 2](https://en.wikipedia.org/wiki/Fallout_2), written by [Martin Janiczek](https://twitter.com/janiczek).
