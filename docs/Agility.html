Agility

25th April 2021 at 11:27pm
SPECIAL

One of the SPECIAL attributes. As such, its value is always an integer between 1 and 10 inclusive.

In-game description:

Coordination and the ability to move well. A high Agility is important for any active character.

Affects:

  • ArmorClass statistic (if the Kamikaze trait is not active)
  • ActionPoints statistic
  • MeleeDamageBonus statistic
  • SmallGuns skill
  • BigGuns skill
  • EnergyWeapons skill
  • Unarmed skill
  • MeleeWeapons skill
  • Throwing skill
  • Sneak skill
  • Lockpick skill
  • Steal skill
  • Traps skill

Is affected by:

  • SmallFrame trait (+1)
  • Gifted trait (+1)
